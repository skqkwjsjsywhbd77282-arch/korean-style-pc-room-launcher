Category system

앱을 폴더에 넣는 방식

설치 후 다운로드 폴더에 Codex PCBang Launcher Categories 폴더가 생성됩니다. 원하는 카테고리 폴더에 바로가기, 실행 파일, URL, 폴더를 넣으면 런처에 표시됩니다.

Uninstall

삭제 방법

  1. Windows 제어판을 열고 검색창에 프로그램 제거를 입력합니다.
  2. 프로그램 제거 또는 변경 화면에서 Korean-style PC Room Launcher를 선택합니다.
  3. 상단의 제거/변경을 누르고 안내에 따라 삭제합니다. 더 정확한 방법은 PDF 매뉴얼 파일에서 확인하세요.

주의: Windows 안전모드에서 Malware Zero 같은 정리 도구를 실행하면 이 프로그램도 삭제될 수 있습니다.

호환성 및 책임 안내

Windows 64비트는 정상 작동을 기준으로 배포하지만, 32비트와 ARM64는 아직 확인이 끝나지 않았습니다.

이 런처의 설치, 사용, 삭제, 호환성 문제에 대한 최종 책임은 사용자에게 있습니다. 지속적인 업데이트는 보장되지 않으며 Windows 64비트 1920x1080 환경에서만 실험했습니다.

오픈소스로 공개될 예정이므로 비공식 업데이트가 진행될 가능성은 있지만, 공식 호환성 보장은 아닙니다.

Dedicated software

자동 업데이트 프로그램

업데이터는 런처와 별도로 설치됩니다. 자동 업데이트를 켜거나, 필요할 때만 선택적으로 확인하도록 고를 수 있습니다.

Codex PCBang Updater

Windows 업데이트 창을 모티브로 한 전용 업데이트 프로그램입니다. 런처 자동 탐색과 수동 탐색을 모두 지원하고, 서명된 업데이트 매니페스트와 SHA256 검증을 통과한 설치관리자만 실행합니다.

  • 자동 모드: Windows 시작 시 백그라운드 확인 후 업데이트가 있을 때만 알림
  • 선택 모드: 사용자가 직접 실행하고 확인 버튼을 눌렀을 때만 검사
  • 보안 모드: 무음 설치 없음, 사용자 확인 후 설치관리자 실행

다운로드

일반 사용자는 LB/PF 설치관리자를 받으면 됩니다. PF는 런처판, LB는 로그인 화면이 포함된 통합판입니다.

이 런처는 아직 대부분 앱과 완벽히 호환되지 않을 수 있습니다.

구분 대상 다운로드 용량 비고
