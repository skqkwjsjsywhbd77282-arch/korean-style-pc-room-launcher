# Code of Conduct

This project uses a simple conduct policy.

## Expected behavior

- Be technical and specific.
- Report bugs with reproduction steps.
- Respect user safety, especially around installer, startup, and uninstall behavior.
- Avoid legal-risk assets and copyrighted UI copying in public contributions.

## Unacceptable behavior

- Uploading malware, suspicious binaries, credentials, or private files.
- Encouraging users to install bug versions on daily-use PCs.
- Hiding functionality in installer scripts or startup tasks.
- Harassment, threats, or abusive comments.

Maintainers may close or remove issues, pull requests, and comments that violate these rules.
