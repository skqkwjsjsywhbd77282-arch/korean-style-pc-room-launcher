Bug Versions

버그버전 기록

지금까지 확인된 문제 기준으로 정리한 기록입니다. 아래 버전은 설치 권장 대상이 아니라 문제 추적용입니다.

버그버전을 개인 컴퓨터에 그대로 설치하면 삭제가 제대로 되지 않거나 자동실행/설정 잔여물이 남을 수 있습니다. 최악의 경우 Windows 복구, 시스템 복원, 재설치가 필요할 수 있으니 실사용 PC에는 설치하지 마세요.
v0.1.1 초기 실험판 기본 런처와 로그인 화면 구조만 확인하던 단계입니다. 카테고리 관리, 설치 후 삭제 안내, 업데이트 로그, 구버전 덮어쓰기 흐름이 아직 정리되지 않았습니다.
v0.1.2 초기 배포/노트북 테스트 문제판 일부 설치 환경에서 로그인 화면이 재부팅 후 뜨지 않는 문제, 제어판 카테고리 미표시, 개인 PC 취향 카테고리 잔존, 스피커/헤드셋 설정 미동작, 업데이트 로그 미표시 문제가 확인되었습니다.
v0.1.3 기능 반영 초기 재빌드판 정렬 탐색기와 아이콘 강제 적용 계열이 들어갔지만, 사이트 다운로드 파일과 릴리스 파일 동기화가 불안정할 수 있고, Store 앱/Steam 게임/일부 바로가기의 아이콘 추출 호환성이 제한적입니다.
v0.1.4 업그레이드 덮어쓰기 미보강판 LB/PF 파일은 다시 만들어졌지만, 기존 구버전이 실행 중인 상태에서 설치할 때 설치관리자가 먼저 런처 프로세스를 닫고 덮어쓰기 설치하는 흐름이 명시적으로 보강되기 전 버전입니다.

GitHub Pages는 정적 HTML/CSS 파일을 게시하고, 각 버전 파일은 GitHub Release 태그와 자산 링크로 연결됩니다.
