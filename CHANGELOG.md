# Changelog

## v0.1.5 - 2026-06-27 15:00 KST

- Added stronger installer overwrite behavior for users with older versions installed.
- Installer attempts to close existing launcher-related processes before upgrade.
- Added public version page with new, old, and bug-version sections.
- Added stronger warnings for bug versions.
- Split normal-user download flow from developer/open-source materials.

## v0.1.4 - 2026-06-27 14:00 KST

- Rebuilt LB and PF x64 installers.
- Updated public download page and release assets.
- Known issue: upgrade overwrite behavior was not yet clearly reinforced.

## v0.1.3 - 2026-06-26 19:00 KST

- Added category tool improvements.
- Added order manager and icon override v2 beta work.
- Added release/source ZIP packaging flow.
- Known issue: shortcut and icon compatibility remained inconsistent across app types.

## v0.1.2 and older

- Early installer and site drafts.
- Known issues included startup/login display inconsistency, category control gaps, and incomplete audio/device settings.
